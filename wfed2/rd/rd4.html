src and srcset

Src is the source of the image. It is the path to the image. Srcset is used for multiple image sizes so that the browser can choose the best image size for the user's device. Doing this saves bandwidth and improves loading times. in srcset, you specify multiple URL sizes for different widths. Then you specify the base src. The markup looks like this: 

                            
                                img srcset="
                                  url size,
                                  url size,
                                  url size
                                 " src="default url"

In the code above, the browser will choose the image that is closest to the user's device width. If the browser doesn't support srcset, it will use the default URL.

Sizes

If you are using the srcset attribute, sizes are a great way to shorten loading times further. You can specify the size of the image in the markup so that it doesn't have to wait for the CSS to be parsed. This is an example of the markup: 

                            
                                img srcset="/img/html/vangogh-sm.jpg 120w,
                                            /img/html/vangogh.jpg 193w,
                                            /img/html/vangogh-lg.jpg 278w"
                                            sizes="(max-width: 710px) 120px,
                                            (max-width: 991px) 193px,
                                            278px"

In the example above, the browser will choose the image that is 120px wide if the viewport is less than 710px. If the viewport is between 710px and 991px, the browser will choose the 193px wide image. If the viewport is greater than 991px, the browser will choose the 278px wide image.

Why art direction?

Art direction is another way to implement responsive images on your website. You use the picture element with source and img elements as the children. The img element is used as a backup in case of compatibility issues or errors. The code would look something like this: 

                            
                                >picture>
                                    >source media="(max-width: 799px)" srcset="ex-480w-close-portrait.jpg" />
                                    >source media="(min-width: 800px)" srcset="ex-800w.jpg" />
                                    >img src="ex-800w.jpg" alt="Person standing holding daughter" />
                                  >/picture>

Using art direction or any markup is useful because it preloads before the CSS and JS are loaded, which means faster loading times.

Summary

It is important to keep your websites responsive as most people are using phones to access the internet now. The two fastest ways of doing this are srcset and art direction. While using either it is critical to have a default case for compatability issues.

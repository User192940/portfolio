Definitions

Selectors are patterns that match elements from the DOM. Properties contain values that style the elements that are assigned to the selectors.

Selectors

:auto-fill is a psuedo-class that allows us to change the styles on an autofilled input from the browser. Default is yellow background and black text.

:required psuedo-class selector allows us to change the styles on any matched element with the required attribute.

Properties

empty-cells: it is a property that tells the DOM how to handle empty cells in a table. Users can decide whether or not to draw borders around the cell or fill the backround. The default value is show. The other value is hide.

place-content: property that combines align-content and justify-content into one property. It is a shorthand property that accepts two values. The first value is the align-content value and the second value is the justify-content value. If one value is set, then both will be the same value.

Summary

There are tons of CSS properties and selectors that can be used to style your website. Selectors are patterns that are used to match elements from the DOM. Properties contain values that are used to style those elements that have been selected. It is important to know what selectors and properties are and where to find the documentation for them.

WP_Post

WP_Post Object

The WP_Post object is an object returned from functions such as get_post(). It contains loads of information about a post.

WP_Post Data

Some variables stored in WP_Post objects are ID, author, type, title, date, excerpt, status, password, comment count, and more. All of this data is easily accessible using the object operator.

WP_Post Role

The role of WP_Post is to give developers easy access to post objects from the database. This allows access to data that will provide a better user experience.

Summary

The WP_Post object contains loads of information about a post. It is returned from the database from functions such as get_post(). Some variables include ID, author, date, excerpt, comment count, and more. The role of WP_Post objects is to provide developers access to objects from the database.

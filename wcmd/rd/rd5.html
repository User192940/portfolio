WordPress Search

WordPress Search Functionality

The search functionality built into WordPress allows visitors to search for content within your website. By default, it searches for the following: Page titles, page content, single-image titles and captions, image alt text, and file names.

URI Role

The URI impacts the search results in WordPress. By default, WordPress search URIs include “?s=search-term”, which can make the URI more difficult to read and understand. The main purpose of the URI is to store the search query input.

Display Active Search Term

We can display the active search term in WordPress by utilizing the query that is inserted into the URL. On submit the form inserts the string into the URL via GET method. We can then use PHP's get_query_var() function to retrieve the string and display it wherever we want.

Summary

The search functionality allows users to search a WordPress website for titles, page content, single-image titles, captions, image alt text, and file names. The URI's role is to store the query in the URI temporarily. Displaying the search query is as easy as using PHP's get_query_var() function to retrieve the string from the URI.

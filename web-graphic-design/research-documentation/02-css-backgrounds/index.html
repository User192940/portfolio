02 Research and Documentation

CSS Background Properties

Position

The background-position property allows you to change the position of the image. The options are top, left, right, center, bottom, percentages, or px. The pixels or percentages measure from the left and then from the top unless otherwise specified.

Attachment

The background-attachment property sets whether a background image is fixed in the viewport or scrolls with its block. The three main values are fixed, scroll, and local. Local fixes the background to the element's contents.

Size

The CSS background-size property sets the size of the element's background image. The image can be stretched, untouched, or constrained.

Edge Offset Value

Allows background images to be positioned relative to the specified edge using 3-4 value syntax. An example is background-position: left 20px top 20px; for positioning 20px from the top left corner.

Multiple Stack Order (location of color value)

The rule for background images is that the first is on top. So if you are using a gradient/color as a background you would most likely want that to be last because you want everything else to be visible.

Summary

The CSS background property contains many properties that allow for an unlimited amount of customization. Some of the most important background properties are position, attachment, and size.

01 Research and Documentation

MDN Web Technology Reference

MDN HTML

The HTML reference page tells website visitors what HTML is and gives links to resources that vary depending on capability.

MDN CSS

The CSS reference page on MDN gives anyone who needs to use CSS everything that they would ever need. The links range from introductory to expert. Additionally, there are tutorials and links for aspiring developers.

Summary

Mozilla Developer Network has many pages of references for a variety of developer languages. The HTML and CSS references contain information about HTML and CSS respectively. Additionally, they contain many links to other resources that go in-depth on the topics.

02 Research and Documentation

DOM Specifications

What is the DOM?

DOM stands for document object model. The DOM represents the HTML of a document on the web. The DOM basically allows people to use code to interact with a document and change things such as text, appearance, and functionality.

DOM Specifications

The people responsible for DOM specifications are the World Wide Web Consortium(W3C).

DOM Implementaion

Browser vendors similarly implement the DOM. Browser vendors utilize the DOM to manipulate web pages. They do this through whatever language suits them best. Many use javascript, but since the DOM is language-neutral, you can use many languages to interact with it.

Summary

The document object model or DOM is the most important API to any web developer. In this documentation, I covered what the DOM is, who made the specifications for the DOM, and DOM implementation.

05 Research and Documentation

AngularJS

According to this the popularity of angular is about a quarter of react which is the most popular right now. Based on what I have seen, angular seems easier to use than jquery. Based on this angular placed 17th out of 20 in performance for javascript frameworks.

ReactJS

React is the most popular JS framework right now. Some pros of react are it is SEO friendly, offers a virtual DOM, and it's easy to learn. Learning to react takes time, but because of its popularity in the industry it is well worth learning the basics right now. This link shows that it has the most searches on stack overflow out of all JS frameworks. Based on this React came in 15th place out of 20. It is slightly faster than Angular but still not the best.

Vue.js

Vue.js is just slightly less popular than angular according to this. Vue is a lightweight and fast framework that offers extensive documentation. Since it has lots of documentation learning the framework will be easier. Some cons are it has comparitively fewer components and a lack of support for large-scale projects. Vue was the 11th-best performer out of the top 20 js frameworks, placing it above react and angular.

Summary

There are tons of JS frameworks on the market right now that boasts ease of use and better performance. Today we looked at some of the most popular and best-performing frameworks. React is the most popular JS framework right now. Vue is a fast and lightweight JS framework. And Angular is an efficient, testable, and open-source framework that allows for succinct yet functional DOM manipulation.

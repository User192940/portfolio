01 Research and Documentation

MDN Web Technology Reference

MDN HTML

The HTML reference page tells website visitors what HTML is and gives links to resources that vary depending on capability.

MDN CSS

The CSS reference page on MDN gives anyone who needs to use CSS everything that they would ever need. The links range from introductory to expert. Additionally, there are tutorials and links for aspiring developers.

MDN DOM

The DOM page on MDN is dedicated to educating people about what a document object model is. It also provides a list of all DOM interfaces and links to pages about them. One would use this page to learn more information about a specific DOM.

MDN Javascript

The MDN javascript page contains all the information anyone would need to know about javascript. Like the other pages, there are numerous links that take you to information with varied levels of complexity. You would use this page if you wanted to dig deeper into a coding concept or refresh your knowledge.

Summary

Mozilla Developer Network has many pages of references for a variety of developer languages. The HTML, CSS, Javascript, and DOM references contain information and tons of links to enhance knowledge of each topic respectively.
